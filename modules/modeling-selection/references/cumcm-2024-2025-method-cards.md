# CUMCM 2024-2025 方法与路线卡

这些是结构到候选路线的映射线索。每条路线必须在当前题目上重新证明数据前提、目标、约束、验证和实现可行性；不得迁移案例的数值、超参数、图表或结论。

## MS-CUMCM-01：多期资源分配的渐进式路线

- **触发**：多期资源分配同时出现轮换/连续性约束、销售或需求上限，后续小问再引入不确定性或相关性。
- **候选链**：确定性约束优化作为透明基线 -> 情景/鲁棒/风险规避扩展 -> 仅在有数据或机制依据时加入相关性。
- **决定测试**：每期可行性、基线比较、情景覆盖、风险参数敏感性。
- **拒绝条件**：不能说明情景、分布、风险偏好或相关性来源时，不把它们写入主模型。
- **来源定位**：用户提供 `math model.zip`，2024 C 题面、附件结构与 `C038.pdf` 首页摘要。

## MS-CUMCM-02：物理前向模型优先于黑箱反演

- **触发**：题面给出可解释机理、观测信号和待估参数，且存在简化/复杂机理分支。
- **候选链**：前向物理模型 -> 信号预处理和初值/特征估计 -> 有界参数反演 -> 由残差和物理条件决定是否升级复杂机理。
- **决定测试**：残差结构、跨条件一致性、参数可辨识性、简化和复杂模型的可比较性。
- **拒绝条件**：复杂模型只因“更高级”被选择，或平滑后没有检查原始信号与失配。
- **来源定位**：用户提供 `math model.zip`，2025 B 题面、4 份光谱表与 `B157.pdf` 首页摘要。

## MS-CUMCM-03：循环决策和时变路径需要状态模型

- **触发**：返工/回收/退货形成循环，或灾害/拥堵等过程使路径代价和可行性随时间变化。
- **候选链**：状态与事件定义 -> 状态传播/价值递推 -> 约束下的决策或时变路径 -> 失效/不可达状态报告。
- **决定测试**：循环终止/收敛、事件顺序、边界状态、多源合流、与静态简化路线的反例比较。
- **拒绝条件**：只按局部成本或静态距离优化，却未表示回流或到达时刻。
- **来源定位**：用户提供 `math model.zip`，2024 B、2025 D 题面及 `B159.pdf`、`D037.pdf` 首页摘要。

## MS-CUMCM-04：连续几何优化宜先分解再复核

- **触发**：多实体、多阶段连续变量导致联合搜索维度高，但题面允许明确的任务/资源分配层。
- **候选链**：单体可行性与覆盖/命中判定 -> 小规模联合基线 -> 有理由的分配-参数分层路线 -> 独立搜索或局部联合复核。
- **决定测试**：判定边界、初值/步长敏感性、分层方案相对小规模联合方案的损失。
- **拒绝条件**：未定义有效判定就优化时长/概率，或以分层便利性代替正确性。
- **来源定位**：用户提供 `math model.zip`，2025 A 题面与 `A196.pdf` 首页摘要。
