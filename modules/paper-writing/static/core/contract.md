# 论文写作契约

论文不是模型和结果的复述，而是从题目要求到模型、真实结果、验证和有限结论的可检查论证。

- `analysis/` 决定题意、要求、变量和数据事实。
- `modeling/` 决定模型、假设、组件、验证计划和不可修改语义。
- `solver/` 决定实际运行、结果、验证、局限和复现证据。
- `paper/` 只能组织、解释和引用这些证据，不能创造第四个事实来源。

每项核心主张必须说明：回答什么要求、证据在哪里、证据支持到什么边界、稿件在哪个位置出现。题目要求未被回答、结果未验证或引用未核验时，保留缺口或回退，不用流畅语言掩盖。

当最终主张的适用范围、风险取舍或创新性表述存在超出证据的多种合理写法时，提出可选表述和证据边界并等待人工确认；不得把写作偏好包装成事实或自动扩大结论范围。
