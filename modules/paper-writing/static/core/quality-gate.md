# 论文交付门禁

始终阻断：上游快照变化、结果来源缺失、伪造数字或引用、题意或模型语义被改写、模板路径越界、终稿仍有占位符。

`draft` 可以保留待补证据、待选文献和待渲染格式，但必须清楚标注。

`final` 需要：

- solver 已允许写作，核心验证和复现完整；
- 每个题目要求有对应回答；
- 每项核心主张为 `supported` 或带边界的 `bounded`；
- 核心引用已核验，图表和数字有真实来源；
- 稿件、来源映射和审计一致；
- 官方模板和规则已记录，目标格式已渲染检查。

不要求固定图表数量、固定引用数量、固定篇幅或固定“创新点”数量，除非官方规则明确要求。
