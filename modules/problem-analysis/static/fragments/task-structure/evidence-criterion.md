# 证据与评价结构检查

适用于 `prediction`、`classification`、`evaluation` 或 `uncertainty-analysis` 标签。只定义评价和信息边界，不推荐方法。

- 明确评价对象、目标方向、单位、比较基准和并列处理。
- 记录观测、决策、结果和评价的时间先后，检查信息泄漏。
- 明确样本、时间窗、群体和空间范围的聚合口径。
- 当同一主体、地点、设备或样本有重复观测、前后记录或成组记录时，登记分析单位、组标识、重复关系和各观测的可用时点。不能默认记录独立；组标识缺失或层级不明时登记为歧义，而不是静默随机拆分。
- 区分随机性、测量误差、缺失、定义不清和不可识别性。
- 用缺失数据、极端值、并列或不同聚合范围挑战结论是否仍可成立。
